CDA-Dehaze: Contrastive and Distillation-assisted Unpaired Real-World Image Dehazing

##################################
Prerequisites
Python 3.6 or above.
##################################
For packages, see requirements.txt.

Getting started
Install PyTorch 1.6 or above and other dependencies (e.g., torchvision, visdom, dominate, gputil).

##################################
CDA-Dehaze Testing
Image test example is provided.

Test the CDA-Dehaze model:
1. please check the options in ./options
2. please run test.py 
The test results will be saved in dic results.
