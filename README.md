# CDA-Dehaze

**Contrastive and Distillation-assisted Unpaired Real-World Image Dehazing**

---

### 🛠️ Prerequisites

- **Python:** 3.6 or above.
- **Packages:** Please refer to `requirements.txt` for a full list of dependencies.

### 🚀 Getting Started

1. Install **PyTorch 1.6** or above.
2. Install other dependencies (e.g., `torchvision`, `visdom`, `dominate`, `gputil`).

> **Note:** For detailed package versions, please check the `requirements.txt` file.
