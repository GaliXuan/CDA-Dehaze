# CDA-Dehaze
Contrastive and Distillation-assisted Unpaired Real-World Image Dehazing

##################################
Prerequisites
Python 3.6 or above.
##################################
For packages, see requirements.txt.

Getting started
Install PyTorch 1.6 or above and other dependencies (e.g., torchvision, visdom, dominate, gputil).
